# TRH Global Calendar

A fully responsive church calendar landing page for **The Reinvention House Ministries Global**, built with **HTML5, CSS3, and JavaScript** and powered by **Teamup Calendar**.

This project serves as the official public-facing calendar for services, programs, leadership meetings, and community outreach events.

---

## Live Site

(https://trh-global-calendar.vercel.app/)

---

## Features

- Dark mode by default
- Fully responsive (mobile, tablet, desktop, TV screens)
- Embedded Teamup calendar (read-only public view)
- Clean, church-branded UI
- Automatic deployment with GitHub Actions
- Analytics-ready
